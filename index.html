BTC Trading Strategies

Experimental Python project for BTC/USDT paper trading with multiple strategy types: scalping, short-term, and long-term. Built using CCXT, pandas, and indicator-based logic.

Status: WIP Mode: Paper trading only

This project is for learning and experimentation. It is not financial advice and should not be used with real money as-is.

Features

  • BTC/USDT paper trading loop using CCXT.
  • RSI-based scalping strategy for 1m / 5m candles.
  • Basic backtesting script to simulate balance changes.
  • Modular structure for adding short-term and long-term strategies.

How to Run (Local)

Clone the repository and install dependencies:

git clone https://github.com/qarnainnazish-a11y/btc-trading-strategies.git
cd btc-trading-strategies
pip install -r requirements.txt

Run a simple backtest:

python backtests/run_backtest.py

Start the paper trading loop:

python main.py
